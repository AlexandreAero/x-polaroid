# x-polaroid

A simple way of archiving X (formerly Twitter) posts.

## Usage
1. Create a JSON file and put every X posts links in an array, for example:
```json
[
  "x.com....",
  "x.com...."
]
```

2. Run (``node .``) and follow the prompts
